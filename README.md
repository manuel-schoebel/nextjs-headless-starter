# Next.js Headless Starter Kit

Stack for next.js headless frontends is documented in confluence:
(Next.js Stack)[https://drausbuettel.atlassian.net/wiki/spaces/EAM/pages/1026949136/Next.js+Stack]

## Setup

```
npm i
npm types:generate
```

## Starting the app

```
npm run dev
```

## Start Storybook

```
npm run storybook
```
